# CS-305
CS 305 - Software Security

## Overview
This repository contains work from CS 305. It includes a completed Artemis Financial security report that demonstrates vulnerability assessment and secure coding practices.

## Reflection

The client in this project was Artemis Financial, a company that needed help improving the security of its software. The main issue was identifying vulnerabilities in their application and making sure sensitive data was protected during communication and storage.

I think I did well at identifying security risks and understanding how different vulnerabilities could affect the system. Coding securely is important because it protects user data, prevents attacks, and keeps systems reliable. Strong security adds value to a company by building trust and avoiding costly breaches.

One of the more challenging parts of the vulnerability assessment was understanding the different types of security issues and how they connect to real-world risks. However, it was also helpful because it showed me how security tools actually work in practice.

To increase layers of security, I implemented encryption using HTTPS and added hashing for data protection. In the future, I would use tools like dependency-check, static analysis tools, and vulnerability scanners to find issues and decide the best way to fix them.

I made sure the application was functional and secure by testing it after making changes and confirming it still ran without errors. After refactoring, I reran security scans and checked for new vulnerabilities to make sure nothing else was introduced.

Some helpful tools and practices I used were dependency-check, secure coding practices, and testing frequently after changes. These will help me in future projects where security is important.

From this assignment, I could show future employers my ability to perform a vulnerability assessment, improve software security, and use tools to identify and fix issues. It demonstrates both my technical skills and my understanding of secure development.
